# fproject

Manage your personal projects directory

## Install

With [fisherman]

```
fisher GustavoKatel/fproject
```

## Usage

```fish
# Opens a project
fproject open PROJECT_NAME

# Creates a new project
fproject new [PROJECT_NAME]

# Set the default project path
fproject path PATH

# Show this help
fproject help
```
